# Release notes data

Markdown files consumed by `/release-notes` and the release skill.

## File naming

`vMAJOR.MINOR.PATCH.md` (e.g. `v1.1.0.md`)

## Frontmatter

```yaml
---
version: "1.1.0"
tag: v1.1.0
date: 2026-07-12
---
```

## Body sections (Russian)

Use `##` headings and bullet lists:

```markdown
## Добавлено

- Описание изменения

## Исправлено

- Описание исправления
```

Generated by `bun scripts/collect-release-notes.ts` or the **release** agent skill.
