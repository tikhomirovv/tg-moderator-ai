## Добавлено

- AI-модерация Telegram-чатов по настраиваемым правилам workspace (OpenAI-compatible LLM)
- Better Auth: регистрация, сессии, workspace (organization), приглашения участников
- Управление ботами, чатами и правилами; per-chat набор правил и silent mode
- Per-rule действия: удаление, предупреждения, бан; whitelist отправителей
- Dashboard workspace: KPI, график активности, recent activity
- История сообщений пользователя в промпте LLM (JSON, retention 100 msg/scope)
- Moderation audit: таблица `moderation_decisions` и страница `/bots/:id/audit`
- Production Docker-образ (GHCR), миграции при старте контейнера, reconcile webhook
- Nitro scheduled tasks: ежедневная retention-очистка (90 дней moderation, 100 msg/scope)

## Исправлено

- Workspace switch без поломки страниц ботов и audit
- Incremental Drizzle migrations (политика no data loss, без `db:reset`)
- Корректные import paths в nested API routes (`decisions`, `logs`)
- Роутинг страницы audit (`[id]/index.vue` вместо конфликта с `[id].vue`)
- Логирование: уровни `info`/`debug`, промпты только в debug
