## Добавлено

- Регистрация, вход и отдельные workspace — у каждой команды свой набор ботов и правил
- Подключение Telegram-ботов, настройка чатов и модерация сообщений с помощью ИИ
- Правила хранятся в библиотеке workspace и назначаются на каждый чат отдельно
- Включение и выключение бота целиком из интерфейса
- Дашборд workspace: ключевые показатели, графики за неделю и лента последней активности
- Приглашение коллег в workspace по электронной почте
- Восстановление пароля через письмо на почту
- Аудит решений модели — можно посмотреть, какое правило сработало и почему
- Страница «Что нового» с историей обновлений приложения
- Белый список в правилах: удобное поле для исключений
- Бот учитывает недавнюю переписку в чате при проверке сообщений

## Исправлено

- При смене workspace страницы обновляются без «залипших» данных
- Тихий режим больше не мешает остальной работе бота
- Предупреждения в Telegram не отправляются, если для правила отключены санкции
- Понятные сообщения об ошибках при входе, регистрации и сбросе пароля
- Корректное открытие страницы аудита бота
- Подтверждение email завершается понятным экраном успеха
