## Добавлено

- Вход в приложение через Telegram — без отдельного пароля и регистрации по почте
- Управление строится вокруг ботов: у каждого бота свои правила, чаты и команда
- Библиотека готовых правил модерации — можно добавлять нужные пресеты по одному (реклама, политика, оскорбления, спам, мошенничество и другие)
- Приглашение коллег в команду бота по коду доступа — владелец выдаёт код, менеджер подключается после входа
- На странице «Что нового» — ссылка на подробности релиза на GitHub

## Исправлено

- Ссылка-приглашение в команду бота сохраняется после входа через Telegram
- В интерфейсе правил и аудита отображаются понятные названия правил, а не технические идентификаторы
- Страница аудита решений модели открывается корректно

## Производительность

- Бот при проверке сообщений учитывает больше недавней переписки в чате
