## Добавлено

- Правила модерации настраиваются отдельно для каждого чата — у одного бота в разных чатах могут быть разные наборы
- У каждого чата своя страница «Модерация»: можно добавлять, менять и удалять правила
- Готовые шаблоны (реклама, политика, оскорбления и др.) добавляются в чат одной кнопкой
- На странице бота видно, сколько правил настроено в каждом чате
- Условия срабатывания правила задаются обычным текстом — без отдельного белого списка

## Исправлено

- Исправлено добавление шаблона правила в чат из каталога
